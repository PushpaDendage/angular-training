Instructions

You need to create a TODO application, In which user will have input a todo and click on add.

After clicking on the button that todo should be added in the list on the rigth,and clicking on the item from list will mark todo as done/undone

To Help you I have added step by step guide
  • Create To div section left/rigth, left will have input and button. and rigth will have the list of todos
  • Button will have have 'onclick' event attached to it clicking on it. will add a child div inside the todolist in right and text of that child div will be input box value
  • Todo list child will have different CSS styles based on there position(even/odd)
  • Now if you click(addEventListener) on the todo(child div) from todo list will mark/unmark the todo(add CSS to the child(line-through))
  • If User click on the button without typing you should not add the todo in the list
